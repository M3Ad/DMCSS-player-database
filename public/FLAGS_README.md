# Flag Images Needed

Add these flag images to the public folder:

1. **puerto_rico_flag.png** - Puerto Rico flag (40x25px recommended)
2. **usa_flag.png** - United States flag (optional)
3. **jamaica_flag.png** - Jamaica flag (optional)
4. **barbados_flag.png** - Barbados flag (optional)

You can download flag images from:
- https://flagpedia.net/ (free, high quality)
- https://www.countryflags.com/

**How to add:**
1. Download the flag image
2. Save it to the `public/` folder
3. Name it exactly as shown above (e.g., `puerto_rico_flag.png`)
4. In Supabase, set the player's `flag` column to `/puerto_rico_flag.png`
